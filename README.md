# cautious-guide

Node.js Service

## Requirements

* [NodeJS](http://nodejs.org) - 4.8 or above

## Install

```sh
git clone https://github.com/raghavmac/cautious-guide
npm install
```
or `yarn` as alternative

## Build

`npm run build` or `yarn build`

## Test

* **Unit** - `npm run test` or `yarn test`
